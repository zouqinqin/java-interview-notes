## Markdown 常用语法

### 标题

# 一级标题
## 二级标题
### 三级标题


### 文字样式

**粗体**
*斜体*
~~删除线~~


### 列表
# 无序列表
- item1
- item2
  - 子item（两个空格缩进）

# 有序列表
1. 第一步
2. 第二步
3. 第三步


### 代码块
```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello");
    }
}
```


### 表格
| 区域 | 线程共享 | 作用 |
|------|---------|------|
| 堆   | ✅ 是   | 存放对象 |
| 栈   | ❌ 否   | 存放栈帧 |


### 链接 & 图片

### 引用 & 分割线
> 这是一段引用内容
> 适合摘录重点原话

## 针对你的笔记场景

| 场景 | 用什么 |
|------|--------|
| 知识点分类 | `##` 二级标题 |
| 重点概念 | **粗体** |
| 代码示例 | 代码块 + 语言标注 |
| 对比内容 | 表格 |
| 每日任务 | 任务清单 |
| 摘录原话 | 引用 `>` |
